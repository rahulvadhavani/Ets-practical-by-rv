# ETS-Practical-By-Rv


### System requirements
- PHP: ^8.2
- Laravel: 11
- Mysql

# Installation

Note : Make suer set Set Env APP_URL based on serve



#### Clone this repo and install composer and setup env by copy example.env
```bash
composer install
```
    
#### Migrate Database
```bash
php artisan migrate
```
#### Seed Database
```bash
php artisan db:seed
```
  
#### Key Generate
```bash
php artisan key:generate
```

#### Storage link
```bash
php artisan storage:link
```

#### Serve the project on localhost
```bash
php artisan serve
```

#### Clear cache once
```bash
php artisan o:c
```
